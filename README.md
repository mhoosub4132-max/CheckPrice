# Discord Price Reader Bot

บอท Discord ที่อ่านราคาจากรูปภาพไอเทม (กรอบ/เอฟเฟกต์/ป้ายชื่อ) โดยใช้ OCR ฟรี (Tesseract.js) แล้วคำนวณราคาขายอัตโนมัติตามสูตรที่ตั้งไว้

## วิธีติดตั้ง

1. ติดตั้ง Node.js เวอร์ชัน 18 ขึ้นไป (ต้องมี `fetch` ในตัว)
2. ติดตั้ง dependency:
   ```
   npm install
   ```
3. คัดลอก `.env.example` เป็น `.env` แล้วใส่โทเคนบอทของคุณ:
   ```
   cp .env.example .env
   ```
4. สร้างบอทที่ [Discord Developer Portal](https://discord.com/developers/applications) แล้วเปิด **Message Content Intent** ในแท็บ Bot ด้วย (สำคัญมาก ไม่งั้นบอทจะอ่าน attachment ไม่ได้)
5. เชิญบอทเข้าเซิร์ฟเวอร์ด้วยสิทธิ์ `Send Messages`, `Read Message History`, `Attach Files`
6. รันบอท:
   ```
   npm start
   ```

## วิธีใช้งาน

พิมพ์ `!price` พร้อมแนบรูปภาพ (แนบได้หลายรูปพร้อมกัน) บอทจะตอบกลับเป็นตารางราคาเหมือนตัวอย่าง

## การตั้งค่าสูตรราคา

แก้ไขได้ที่ `src/config.js` ทั้งหมด:

- `priceTiers` — ตารางราคาร้าน (กรณีมีไนโตร) แบ่งตามช่วงราคาดิสคอร์ด
- `noNitroSurchargePerItem` — ค่าบวกเพิ่มต่อชิ้นถ้าลูกค้าไม่มีไนโตร
- `ocr.pricePattern` — regex ที่ใช้ดึงตัวเลขราคาจากข้อความที่ OCR อ่านได้

## ข้อจำกัดที่ควรรู้ (สำคัญ)

เพราะใช้ OCR ฟรีแทน AI vision จึงมีจุดที่ต้องระวัง:

1. **ความคมชัดของรูปมีผลมาก** — รูปเบลอ/ตัวหนังสือเล็กเกินไปจะอ่านผิดพลาดได้ ควรให้ user ส่งรูป screenshot แบบเต็มความละเอียด ไม่ใช่รูปที่ถูกบีบอัด/ย่อขนาดจากแอปอื่น
2. **แยกราคาขีดฆ่ากับราคาลดไม่ได้ 100%** — OCR อ่านได้แค่ตัวหนังสือ ไม่รู้ว่าอันไหนถูกขีดฆ่า โค้ดนี้แก้ปัญหาเบื้องต้นด้วยการ "เลือกราคาที่ต่ำที่สุดที่เจอ" ซึ่งถูกต้องในเคสส่วนใหญ่ (เพราะราคาลดมักต่ำกว่าเสมอ) แต่ถ้าอยากแม่นขึ้นจริง ๆ ต้อง crop เฉพาะพื้นที่ราคาปัจจุบันก่อนส่งเข้า OCR (ตำแหน่งจะขึ้นกับ layout ของรูปที่ user ส่งมา)
3. **ถ้าต้องการความแม่นยำสูงสุด** — แนะนำให้เปลี่ยนไปใช้ Claude API (vision) แทน Tesseract ในอนาคตถ้ามีงบ เพราะอ่าน layout ที่หลากหลายได้แม่นกว่ามาก โดยไม่ต้อง crop เอง
